# 💄 BeautyHub 

> **Status:** 🚧 Work in Progress (Jan 2026 – Present)

BeautyHub is a responsive full-stack e-commerce platform dedicated to beauty products. Built with the MERN stack and utilizing a RESTful architecture, this application allows users to browse a product catalog, manage their shopping carts, and maintain personal profiles with secure authentication. 

## 🚀 Tech Stack

**Frontend:**
* React.js
* Tailwind CSS (for responsive UI design)

**Backend:**
* Node.js
* Express.js
* JSON Web Tokens (JWT) for secure authentication
* RESTful APIs

**Database:**
* MongoDB

## 💻 Running the Project Locally

To test or contribute to BeautyHub locally, follow these steps:

### Prerequisites
Make sure you have Node.js and MongoDB installed on your machine.

### Getting Started

1. **Clone the repository:**
```bash
   git clone https://github.com/vindyacharuni/BeautyHub .git
```
2. **Install Backend Dependencies:**
   ```bash
   cd backend
   npm install

3. **Install Frontend Dependencies:**
   ```bash
   cd frontend
   npm install
4. **Environment Variables:
Create a .env file in the backend directory and add the following keys:**
```bash
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
```
5. **Start the Development Servers:**
   ```bash
   # Run the backend server (from the backend directory)
   npm start
   
   # Run the frontend application (from the frontend directory in a new terminal)
   npm run dev
   ````
